Frequently Asked Questions
  The app isn't saving my progress
    Make sure you have storage permissions enabled in Settings.
  How do I flag a mine?
    Long press on any unrevealed cell to place or remove a flag.
  What is Hex Mode?
    A reimagined version of minesweeper on a hexagonal grid where every cell has 6 neighbors instead of 8.
  What is Fog of War?
    A mode where the board is hidden in darkness — cells light up as you reveal them.
